API

Register request

Used to create a new user with POST method.

Path: /users/register/

JSON example: 

{
    email: email,
    password: password,
}

Login request

Used to log a user in with POST method.

Path: /users/login/

JSON example: 

{
    email: email,
    username: username,
    password: password,
    passwordVerif: passwordVerif
}
